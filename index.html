📖 Setup Guide — Read First
📱 STEP 1 — WhatsApp via CallMeBot (Free)
  1. Open WhatsApp on your phone
  2. Send a message: I allow callmebot to send me messages
  3. Send it to this number: +34 644 71 11 14
  4. You will receive an API key reply within 1 minute
  5. Copy that API key and paste it in "CallMeBot Key" field above
  6. Enter your WhatsApp number with country code in "WhatsApp No." field (e.g. 919876543210 for India)
📧 STEP 2 — Email via EmailJS (Free — 200 emails/month)
  1. Go to emailjs.com and create a free account
  2. Click "Add New Service" → Choose Gmail/Outlook → Connect your email
  3. Note your Service ID (e.g. service_abc123)
  4. Click "Email Templates" → "Create New Template"
  5. Set template body as: {{message}} and subject as: Bhairav Security Alert
  6. Note your Template ID (e.g. template_xyz456)
  7. Go to "Account" → copy your Public Key
  8. Enter all 3 values in the fields above along with your email address
⚡ After setup: Click "✏️ Edit", enter all keys, click "💾 Save Settings", then click "🧪 Test WhatsApp + Email Now" to verify everything works.
